## Server (FastAPI)

**Install dependencies:**
```sh
cd server
uv pip install -r pyproject.toml
```

**Run development server:**
```sh
uv run fastapi dev
```

---

## Client (React/TypeScript)

**Install dependencies:**
```sh
cd client
npm install
```

**Run development server:**
```sh
npm run dev
```
